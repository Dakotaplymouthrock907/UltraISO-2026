## 💿 Working with ISO Files

The software uses a virtual drive to handle disc images. This feature allows your computer to treat an ISO file like a physical disc located in a drive.

### Opening an Image File
Use the File menu to open your ISO file. The right-hand pane shows the contents of the image. You can drag and drop these files to your desktop to extract them.

### Editing an Image
Add or remove files inside the ISO window. Click Save to apply changes to the existing file. Consider using the Save As option if you wish to keep the original file unchanged.

### Creating a Bootable Disc
Insert a blank CD, DVD, or a USB drive. Select your image file, then click the Bootable menu. Select Write Disk Image. Choose your drive from the list. Click Write to create a bootable medium. This proves useful if you want to install an operating system from a USB stick.

## 🧪 Troubleshooting

If you encounter issues, review these common solutions.

### Software Does Not Start
Check if your antivirus software blocks the application. Add an exclusion for the installation folder if necessary. Ensure you launched the installer with administrator privileges.

### Image File Errors
Verify that your ISO file is not corrupt. Try redownloading the file if the error persists. Ensure you have enough disk space to extract large contents from an image.

### Virtual Drive Not Found
Restart your computer after installation. The drivers for the virtual CD/DVD-ROM usually require a system reboot to function correctly. If the issue remains, open the application settings and ensure the virtual drive count is set to at least one.
